Glosario

AJAX

Técnica que permite enviar y recibir datos de un servidor de forma asíncrona.

API

Conjunto de reglas que permite la interacción entre diferentes software y aplicaciones.

Bootstrap

Framework CSS que permite diseñar sitios web responsivos y móviles.

CMS

Sistema que permite a los usuarios crear y administrar contenido web.

Cookie

Pequeño archivo que almacena información sobre la visita de un usuario a un sitio web.

DOM

Modelo de objetos que representa la estructura de un documento HTML o XML.

Favicon

Pequeño ícono que se muestra en la pestaña del navegador junto al título de la página.

Git

Sistema de control de versiones que permite gestionar y rastrear cambios en el código fuente.

HTML

Lenguaje de marcado estándar para crear páginas web.

HTTP

Protocolo de comunicación utilizado para la transmisión de datos en la web.

HTTPS

Versión segura de HTTP que utiliza encriptación para proteger la comunicación entre el cliente y el servidor.

JSON

Formato ligero de intercambio de datos que es fácil de leer y escribir para humanos.

Minificación

Proceso de eliminar caracteres innecesarios del código fuente para reducir su tamaño.

Prototipo

Modelo preliminar de una página web o aplicación que simula su funcionamiento y diseño.

Responsive Design

Enfoque de diseño web que hace que las páginas se vean bien en una variedad de dispositivos y tamaños de pantalla.

Sass

Preprocesador CSS que permite utilizar variables y anidamiento.

SEO

Conjunto de prácticas que mejoran la visibilidad de una página web en los motores de búsqueda.

UI/UX

Diseño de la interfaz de usuario (UI) y la experiencia del usuario (UX) para mejorar la usabilidad de una aplicación.

Webpack

Herramienta que permite agrupar y optimizar archivos JavaScript y otros recursos.
